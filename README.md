# A7 餐廳清單擴充 CRUD 功能 

# 功能
1. 顯示所有餐廳
2. 瀏覽詳細資料
3. 新增餐廳
4. 修改餐廳資料
5. 刪除餐廳

# 開始使用
1. 安裝 node.js 與 npm
2. 專案 clone 到本地
3. 開啟資料夾，輸入
4. 繼續輸入 npm run dev
5. 瀏覽器輸入localhost : 3000
6. 結束請輸入Ctrl + C

# 使用工具
1. Express v4.17.1
2. Express-handlebars v 4.0.2
3. Mongoose v5.13.15
4. Body-parser v1.20.1
5. Node.js v 16.17.0
6. Front-awesome
